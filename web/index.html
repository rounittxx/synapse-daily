How it works

Built with Python, sentence-transformers, and Claude API

01

Fetch

Every morning, the pipeline pulls fresh articles from 10+ RSS feeds — Hugging Face, arXiv, MIT Tech Review, VentureBeat, DeepMind, and more.

→
02

Rank

A local all-MiniLM-L6-v2 sentence-transformer scores each article's AI/ML relevance and removes duplicates using cosine similarity.

→
03

Summarise

The top-ranked articles go to Claude, which writes concise summaries, key takeaways, and an editorial closing note — no filler, just signal.

→
04

Deliver

A clean HTML email lands in your inbox every morning at 7 AM UTC via GitHub Actions. Works in every email client.
